Apply various rules to retrieve category data.

Inherited from: \XML_Feed_Parser_Type::getCategory()

There is no single way of declaring a category in RSS1/1.1 as there is in RSS2 and Atom. Instead the usual approach is to use the dublin core namespace to declare categories. For example delicious use both: PEAR and: to declare a categorisation of 'PEAR'.

We need to be sensitive to this where possible.
